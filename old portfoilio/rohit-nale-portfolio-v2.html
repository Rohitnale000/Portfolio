SYS-001 · Location-Based Dating Platform
Cordia — Geo-Matching Engine
LIVE
Problem
Real-time discovery of nearby users required precise geo-calculations at scale — standard SQL distance queries weren't performant enough for live matching.
Architecture
GraphQL API over Node.js + Prisma. Haversine formula embedded in query resolvers for accurate Earth-curve distance. Modular domain-based schema design.
Security
JWT authentication, field-level data access control, sensitive profile data isolated by RBAC patterns. Zero-trust resolver design.
Impact
Shipped complete user lifecycle: onboarding → discovery → matching → filters. Precision geo-matching with sub-50ms query response.
